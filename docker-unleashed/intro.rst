**********
Introdução
**********

Seja bem vindo ao curso *Docker Unleashed*, cujo objetivo é capacitar o aluno a:

 * Destacar as vantagens e desvantagens do Docker e conhecer os termos correlatos a esta tecnologia;
 * Disponibilizar aplicações em forma de microsserviços;
 * Gerenciar aplicações e o ecossistema Docker de forma eficiente;
 * Reconhecer as melhores práticas de administração e segurança do serviço.

Este curso foi programado para ser ministrador em 35 Horas-Aula, divido no período de uma semana de aplicação, ou seja, 7 horas por dia.

Conteúdo Programático
#####################

Cada dia de curso está disposto em uma aula, com os seguintes tópicos:

 :doc:`Aula 1: Aula 1: Docker - Imagens e Contêineres <aula1>`
   - Diferenciação entre virtualização e virtualização de contêineres;
   - Introdução a Microserviços e TI 'Imutável': Ciclo de vida de contêineres.
   - Instalação do Serviço Docker;
   - Criação e gerenciamento de imagens;
   - Volumes.


 :doc:`Aula 2: Docker: Docker: Funcionamento e Gerenciamento <aula2>`
   - Rede;
   - Log Drivers;
   - Docker Compose


 :doc:`Aula 3: Gerenciamento de recursos e Segurança <aula3>`
   - Docker Registry;
   - Posix CAPS, SecComp, AppArmor, Selinux;
   - Autorização e segurança das variáveis de ambiente dos contêineres;


 :doc:`Aula4: Gerenciamento de Volumes & Monitoramento <aula4>`
   - Distribuição de Volumes: BTSync, Flocker;
   - Monitoramento: CAdvisor, Scout, DataDog, SysDig Cloud.


 :doc:`Aula 5: Cluster, Gerenciamento <aula5>`
   - Cluster: Docker Swarm;
   - Gerenciamento: Rancher


As aulas serão ministradas de forma coordenada, onde instrutor e aluno realizam as mesmas ações em suas estações de trabalho de acordo com o andamento da aula.

Requisitos ténicos para realização do Curso
###########################################

Para a realização deste curso é impreterível a utilização de um computador que contenha o sistema operacional GNU/Linux, cuja arquitetura seja 64 bits.

Requisitos de conhecimento prévio
#################################

Para a realização do curso, é necessário que o aluno possua conhecimentos **intermediários** no Sistema Operacional.

**Este curso não é recomendado para aqueles que ainda não tiveram contanto com o Sistema Operacional Linux ou o que não o operam via CLI.**
